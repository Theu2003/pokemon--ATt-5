## Aluno: Mateus Martins Peres

## Descrição da Atividade

Atividade 05 da disciplina de Web. Site com tema de cartas raras do Pokémon TCG, desenvolvido para praticar HTML e CSS.

## Páginas

- `index.html` — Página inicial com carta em destaque, lista e vídeo
- `galeria.html` — Galeria com as cartas raras
- `contato.html` — Formulário de contato com 5 controles
- `sobre.html` — Informações sobre o projeto e o autor
